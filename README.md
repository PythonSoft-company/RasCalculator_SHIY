Добрый день это проект расширенный калькулятор\
Какая задача у этого калькулятора?

1. Решать обычные задачи из математик
2. Решать уравнения
3. Решать задачи  по статистике(Среднее значение, Медиана, Максимальное значение, Минимальное значение, Размах, Дисперсия)
4. Решать задачи по триногреметрии
5. Решать задачи по дробям\
Имеется история и отдельное поле для показа ошибок и система логирования (Для предоставления неизвестной ошибки Разработчику)\
Сейчас написан на Python+Tkinter+Matplotlib, производится изменение Tkinter на PyQt6, срез(Freezing) производится на через cx_Freeze
